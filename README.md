# AI Projects — CS50 AI (Harvard)

This repository contains implementations of core AI concepts using Python, covering search, reinforcement learning, probabilistic reasoning, and constraint satisfaction.

These projects were built as part of Harvard’s CS50 AI course and further explored to strengthen foundational understanding of artificial intelligence systems.

---

## 🔗 Highlight Project

👉 **Traffic Sign Interpretation System (Deployed)**  
https://huggingface.co/spaces/Arghya09/traffic-sign-classifier  

An end-to-end AI system that classifies and interprets traffic signs using a CNN model, with real-time inference through a web interface.

---

## 📂 Projects

### 1. Traffic Sign Classification (CNN - TensorFlow/Keras)
- Built a convolutional neural network to classify 43 traffic sign categories  
- Used OpenCV for preprocessing  
- Multi-class classification using softmax  

---

### 2. Reinforcement Learning — Nim Game
- Implemented a Q-Learning agent  
- Learns optimal strategies through self-play  

---

### 3. Probabilistic Reasoning — Heredity
- Bayesian network to compute genetic inheritance probabilities  
- Handles uncertainty and mutation effects  

---

### 4. Crossword Puzzle Solver (CSP)
- Solved using backtracking and constraint propagation  
- Efficient constraint satisfaction implementation
