# AI Projects – CS50 AI (Harvard)

This repository contains implementations of core AI concepts using Python.

## Projects

### 1. Traffic Sign Classification (CNN - TensorFlow/Keras)
- Built a convolutional neural network to classify 43 traffic sign categories
- Used OpenCV for preprocessing
- Multi-class classification using softmax

### 2. Reinforcement Learning – Nim Game
- Implemented a Q-Learning agent
- Learns optimal strategies through self-play

### 3. Probabilistic Reasoning – Heredity
- Bayesian network to compute genetic inheritance probabilities

### 4. Crossword Puzzle Solver (CSP)
- Solved using backtracking and constraint propagation

## Tech Stack
